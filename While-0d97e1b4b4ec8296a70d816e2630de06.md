
> Um _while loop_ executa repetidamente um bloco de código **enquanto uma condição for verdadeira**. Antes de entrar no loop, você geralmente define uma variável de controle.


```assembly
i = 1
while i < 6:
    print(i)
    i += 1
```


---


### Interrompendo o loop `break`


Você pode usar `break` para sair imediatamente do loop, **mesmo que a condição ainda seja verdadeira**. 


```python
i = 1
while i < 6:
    print(i)
    if i == 3:
        break
    i += 1
```


---


### Pular a iteração atual  `continue` 


`continue` interrompe apenas a iteração atual e **vai direto para a próxima**, sem executar o restante do corpo do loop nessa rodada.


```python
i = 0
while i < 6:
    i += 1
    if i == 3:
        continue
    print(i)
```


---


### `else` no _while_


O `else` pode ser usado após um _while_ para executar um bloco **quando a condição se tornar falsa**. 


💡 _Importante:_


    A parte do `else` **não roda se o loop foi encerrado com** **`break`**.


```python
i = 1
while i < 6:
    print(i)
    i += 1
else:
    print("i is no longer less than 6")
```


---


### 📌 Conceitos chave


| Conceito         | O que faz                                                  |
| ---------------- | ---------------------------------------------------------- |
| `while condição` | Executa código enquanto condição for verdadeira            |
| `break`          | Sai totalmente do loop                                     |
| `continue`       | Pula o restante do código e vai para a próxima verificação |
| `else`           | Executa depois do loop quando a condição é falsa           |
