
O `match` é uma estrutura de controle em Python usada para executar diferentes blocos de código com base em diferentes condições, funcionando de forma semelhante a um _switch/case_. 


🔹 **Por que usar**


Ele facilita substituir várias estruturas `if..elif..else`, tornando o código mais claro e legível quando se depende da comparação de um valor com muitos casos possíveis. 


---


### **Sintaxe Básica**


```python
match expressão:
    case valor1:
        bloco de código1
    case valor2:
        bloco de código2
    case valor3:
        bloco de código3
```


💡 A expressão é avaliada **uma vez** e comparada com cada `case` em ordem até encontrar uma correspondência.


---


### **Caso Padrão (Default)**


👉 Use `_` como `case` final para tratar qualquer valor que não combine com os anteriores — funciona como o `else` ou `default`: 


```python
match valor:
    case 1:
        print("Um")
    case _:
        print("Outro")
```


---


### **Combinar Múltiplos Valores**


👉 O operador `|` (pipe) atua como um _OR_ no mesmo `case`:


```python
case 1 | 2 | 3:
    print("Um, dois ou três")
```


Isto permite verificar vários valores em uma única linha.


---


### **Guards com** **`if`**


👉 Também é possível adicionar uma condição extra após um `case` com `if` (_guards_) para refinar quando aquele bloco deve ser executado:


```python
case x if x > 10:
    print("Maior que 10")
```


A execução depende de a expressão e a condição adicional serem satisfeitas. 


---


### **Resumo dos Pontos Principais**


✅ Substitui estruturas longas de `if..elif..else` para valores distintos


✅ Suporta casos padrão com `_`


✅ Permite combinar múltiplos valores com `|`


✅ Permite condições extras com `if`
