> 📌 Um **loop** **`for`** em Python é usado para **iterar sobre uma sequência** — como uma lista, tupla, conjunto (_set_), dicionário ou string — e executar um bloco de código para cada item dessa sequência. 


---


### 🔁 Como funciona

- **Percorrer elementos de uma lista**

    Exemplo básico: cada item da lista é atribuído à variável e impresso. 

- **Strings também são iteráveis**

    Cada caractere da string pode ser visitado dentro de um loop. 


---


---


### 🚫 Controle de Fluxo

- **`break`** — sai do loop antes de terminar todas as iterações.

---


### 🔄 Loops Aninhados (_Nested Loops_)


Você pode colocar **um** _**for**_ **dentro de outro**.


No exemplo comum, para cada item da primeira sequência, o loop interno percorre todos os itens de outra. 


---


### 🪪 `pass` em um _for_


Se você precisa de um loop sem corpo (por exemplo, enquanto estrutura está sendo desenvolvida), use `pass` para evitar erros.


---


## 📌 Observações rápidas

- O _for_ em Python **não usa índice explicitamente** como em outras linguagens (a não ser que você utilize `range()` ou `enumerate()`), pois ele vai diretamente sobre os elementos da sequência.
