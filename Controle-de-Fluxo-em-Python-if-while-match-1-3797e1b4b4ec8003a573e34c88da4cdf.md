<summary>❌ **Python Loop Control Statements**</summary>
1. **Break:** Terminated the flow of the loop statement and executes the next statement outside the loop.
  
2. **Continue:** It is used when we need to skip the execution of the remainder of statements in the loop and continue from the start.
   
3. **Pass**: It is used when we need some statements syntactically but does not want to put any statements.
